## Running Locally

### Option 1: Direct File Open
Simply open `index.html` in your web browser.


## Customization

### Colors
Edit the CSS variables in `css/style.css`:
```css
--primary-color: #FF8C42;
--secondary-color: #FFA500;
--dark-color: #2C3E50;
--light-color: #ECF0F1;
```

### Content
Edit sections in `index.html` with your actual content and images.

### Dog Profiles
Update dog cards in the adoption section with real dog information.
